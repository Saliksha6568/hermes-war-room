## 🧩 Understanding agent behavior

The hermes-war-room handles orchestration for your agents. You may see agents running in parallel. This means they perform multiple steps at once to save time. You do not need to intervene as the agents finish these steps. Observe the status icons. A green light means the task proceeds as expected. A yellow light suggests the agent waits for input or network data. A red light indicates a blocked task. If you see a red light, click the agent and check the error log at the bottom of the screen.
